# Contributing

Thanks for considering contributing to this project! Contributions of all kinds are welcome, including bug reports, fixes, improvements, and documentation.

## How to contribute

* Fork the repository
* Create a feature branch from `main`
* Make your changes with clear, focused commits
* Open a pull request with a short description of what and why

If your change is non-trivial, please consider opening an issue first to discuss it.

## Code style and quality

* Keep changes minimal and focused
* Follow existing code style and conventions
* Add comments or documentation where it improves clarity

## Licensing

By submitting a contribution, you agree that your contribution will be licensed under the same license as this project.

You also confirm that you have the right to submit the contribution and that it does not infringe on third-party rights.

## Questions

If you are unsure about anything, feel free to open an issue or start a discussion.
